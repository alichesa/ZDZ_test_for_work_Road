# word_road_map

 # 一、核心理念
 明确学习目标，将知识体系分为**通用基础**和**岗位专项**两部分，系统化构建竞争力。

 # 二、知识体系分解
 1. 通用基础部分 -- 这是所有技术岗位的基石，必须牢固掌握。
 
 - C++ 语言深度:入门与算法思想：建议先通读《Hello 算法》,这是理解高级八股的重要铺垫。  ```https://www.hello-algo.com/chapter_hashing/hash_map/```
             进阶八股与深度：在掌握基础后，需深入语言特性、内存管理、面向对象、模板元编程等。可通过**牛客等平台刷面经**，**从“历年真题”中反向查漏补**缺。
 - 计算机核心基础（计算机网络、操作系统、数据库）：强烈推荐小林coding的《图解网络》。```https://xiaolincoding.com/network/1_base/what_happen_url.html#%E5%AD%A4%E5%8D%95%E5%B0%8F%E5%BC%9F-http```

 - 数据结构与算法 (Leetcode)
  第一阶段（语法与基础）：结合《Hello 算法》中的算法实现，刷完Leetcode前100道简单题，巩固C++语法和基础数据结构应用。
  第二阶段（系统训练）：跟随《代码随想录》的题目分类和节奏，进行系统性、专题性的刷题训练。
  第三阶段（面试准备）：面试前，必须确保Hot 100题目烂熟于心，建议反复练习2-3遍，达到熟练手写的程度。

  2. 岗位专项部分 (以AI/C++为例)
在夯实基础后，针对目标岗位进行纵深学习。
AI/大模型方向：需要学习机器学习基础、深度学习框架、以及当前大模型的原理、应用和优化技术。
高性能计算/超算方向：需要了解并行计算、CUDA编程、系统性能调优等知识。
持续关注：主动寻找并研究相关的Roadmap，紧跟业界技术发展趋势。
3. 项目经验积累
“Talk is cheap. Show me the code.” 项目是知识融会贯通的试金石。
中级挑战项目：如果有志于C++后端，强烈推荐研究Web Server项目（牛客可搜到）。该项目综合性强，几乎涵盖网络编程、并发处理、数据库连接等C++后端核心技能。
开源社区实践：积极参与Infinitensor等开源社区举办的活动（如单片机、Docker、CUDA相关项目）。这些项目难度适中，成果明确，是写入简历的绝佳材料，能极大展现你的工程能力和协作精神。
三、执行力与氛围
最后也最关键的一点：在牛客等多刷面经、多参与讨论。找到那些优秀且目标一致的同行者，学习他们，追赶他们，甚至尝试超越他们。在积极的氛围和良性的竞争中，你的成长速度会远超独自摸索。
